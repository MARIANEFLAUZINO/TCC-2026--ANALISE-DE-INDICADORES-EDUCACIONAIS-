# 📚 Educação Básica e Desenvolvimento Econômico no Brasil

Dashboard interativo desenvolvido como Trabalho de Conclusão de Curso (TCC),
analisando disparidades regionais na educação e seu impacto socioeconômico.

## 🎯 Objetivo
Investigar a relação entre o desempenho do IDEB e o PIB per capita dos
municípios brasileiros, e entre níveis de escolaridade e salário médio nos estados.

## 🗃️ Dados Utilizados
- **IDEB** — ciclos 2013, 2015, 2017 e 2019 (INEP)
- **PIB per capita** — IBGE (5.295 municípios)
- **PNAD** — escolaridade e salário médio por estado

## 🔍 Metodologia
- Estatística descritiva
- Análise de clusters para identificação de padrões regionais
- Visualização interativa via Power BI

## 📈 Principais Resultados
- Municípios com maior IDEB tendem a apresentar maior PIB per capita
- Trabalhadores com maior escolaridade possuem salários médios
  significativamente superiores
- Há progressão salarial consistente conforme o avanço educacional

## ⚠️ Observação sobre os dados
Foram utilizados dados de **5.295 municípios** dos 5.570 registrados pelo IBGE.
Os 275 municípios ausentes não possuíam dados completos disponíveis
nas fontes consultadas para o período analisado.
